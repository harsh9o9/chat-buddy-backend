# Chat Buddy Backend
Backend for chat buddy realtime chatting website, build using socket.io and Express.js

## Technologies:
- Socket.io
- Express.js
- Node.js
- Express validator
- jsonwebtoken
- mongoose

## Startup:
- Need git, node, npm installed
- Clone the repo `git clone <repo-link>`
- Install dependencies `npm install`
- Run server `npm start`
